# Ling 基本语法

## 注释
`//` 是注释
``` ling
// 注释, 写给自己或者别人看的内容, 不会影响代码运行
```

## 命名空间
``` ling
// 引用命名空间
use LingIO;

// 声明命名空间
// ns 之下的代码都属于这个命名空间, 命名空间不可嵌套
ns LingMain;

// Main 方法是程序开始执行的地方
// 每个程序都必须有且只有一个 Main 方法
static void Main() {
    // 调用 Print 方法
    // 把 "Hello World!" 作为参数传入
    Print("Hello World!");
}
```
